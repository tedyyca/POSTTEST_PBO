## Deskripsi Program
Program ini adalah aplikasi manajemen stok sepatu olahraga berbasis CLI (Command Line Interface). Program ini mengelola data sepatu menggunakan ArrayList yang bersifat dinamis, sehingga pengguna dapat menambah, melihat, mengubah, dan menghapus data stok dengan mudah selama program berjalan.

## Fitur Utama
1. **Tambah Stok (Create)**: Memasukkan data sepatu baru dengan sistem pilihan merk (1-5) dan kategori (1-3) untuk memastikan akurasi data.
2. **Lihat Semua Sepatu (Read)**: Menampilkan seluruh daftar sepatu yang tersedia beserta detail harga, stok, dan kategorinya.
3. **Ubah Data Sepatu (Update)**: Memperbarui informasi harga sepatu pada daftar yang sudah ada berdasarkan nomor urut.
4. **Hapus Sepatu (Delete)**: Menghapus data sepatu secara permanen dari daftar stok.
5. **Exit**: Menghentikan eksekusi program secara aman.
